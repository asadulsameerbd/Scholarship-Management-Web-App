# 🎓 Scholarship Management System

A full-stack web application designed to simplify scholarship discovery, application, and management. This platform supports multiple user roles including Students, Moderators, and Admins.

---

## 🚀 Features

### 👨‍🎓 Student
- Browse and search available scholarships
- Apply for scholarships بسهولة
- View applied scholarships from dashboard
- Update profile information

### 🛠️ Moderator
- Add new scholarships
- Manage (edit/delete) scholarship listings
- Monitor applications
- Update profile

### 👑 Admin
- Full system access
- Manage all scholarships
- Manage users (students & moderators)
- Manage reviews and applications
- Update admin profile

---

## 🧑‍💻 Tech Stack

**Frontend:**
- React.js
- Tailwind CSS
- DaisyUI

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Authentication:**
- JWT (JSON Web Token)
